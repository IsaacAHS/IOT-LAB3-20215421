En este lab, se solicito realizar una aplicación de trivia. Para ello se diseño la vista principal el cual contendría dos despelgables: categoría y dificultad, de acuerdo a lo solicitado. Un campo para ingresar el numero de preguntas(mayor a 0) un botón de verificar conexión con el id "btnCheckConnection" donde revisa si el dispositivo cuenta con internet, si cuenta, se habilita el botón "Empezar", sino hay, no se habilitara tal botón. Una vez llenados todos los campos(categoría, dificultad y cantidad) pasará a la siguiente actividad que es el juego en sí. Para esta parte, se recibe los datos del MainActivity, y se calcula el tiempo disponible para ello. En esta parte lo mas importante fue la conexión a la API publica Open Trivia Database, para ello usé OkHttp para descargar las preguntas escogidas por el usuario. Además un temporizador desde cuando carga la pantalla, si se termina el tiempo, se termina el juego. Las respuestas correctas, incorrectas y no respondidas, tiempo utilizado, categoría, dificultad son enviadas al activity de estadísticas donde son mostradas con TextViews. Además añadí un grafico pastel(CustomChartView) para una mejor visualización del score. El activity Question modela los datos de la pregunta, sirve para manejar y pasar preguntas. Solo se uso hilos internos como "CountDownTimer" y el de llamadas de red "OkHttp" . Los demás activities no requieren un uso significativo de los recursos. Solo se muestran datos, se modifica la interfaz grafica, ejecuciones simples como cambiar de actividad (intent).Esta app fue terminada gracias a Claude y ChatGPT xd.
